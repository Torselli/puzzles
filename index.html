☁️ Google Drive

Seus dados são salvos automaticamente no Drive e sincronizados entre todos os seus dispositivos.

Passo 1
Acesse console.cloud.google.com → crie um projeto novo (ex: minha-colecao)
Passo 2
Ative as APIs: Google Drive API e Google Picker API
Menu → APIs e Serviços → Biblioteca → buscar e ativar cada uma
Passo 3
Crie credenciais: APIs e Serviços → Credenciais → Criar credenciais → ID do cliente OAuth 2.0
Tipo: Aplicativo da Web → adicione sua URL do GitHub Pages em "Origens JavaScript autorizadas"
Passo 4
Também crie uma Chave de API (Credenciais → Criar credenciais → Chave de API)
